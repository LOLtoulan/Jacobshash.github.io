Java线程通信之生产者和消费者

一、生产者和消费者之间的关系

生产者消费者模式：就像卖包子，现在店里有100个包子，卖给顾客，顾客就是消费者，但现在包子店太火了，100不够，这是要保证店里有100包子，就需要生产新的包子，这时，店家就是生产者，这就是生产这消费者模型。

也就是说生产者消费者模型，是不同线程（生产者，消费者）操作同一对象、同一资源（包子）。

在撸代码之前，我们需要整理下如何写代码，

首先，我们要创建”包子类”，也就是资源类，

然后我们要创建”生产者”和”消费者“

最后我们创建测试类，

也就是说我们要创建四个类

我们跟着思路来写：

资源类：

public class Student {
    String name;
    int age;
    String sex;
}

生产者类：

public class SetThread implements Runnable {
    @Override
    public void run() {

        student.age = 18;
        student.sex = "female";

public class GetThread implements Runnable{

public class StudentDemo {

所以现在存在了线程安全问题，既然知道了原因，所以解决就变得很简单了。

所以需要加锁，但加锁也要注意，我们只对生产者加锁，或只对消费者加锁都是没有意义的，需要同时加锁，且不同线程加的锁必须是同一把。

修改如下：

public class Student {
    String name;
    int age;
    String sex;
}
public class SetThread implements Runnable {



    private Student student;
    private int x= 0;
    public static Object obj = new Object();

    public SetThread(Student student) {
        this.student = student;
    }
    @Override
    public void run() {

        while (true) {
            synchronized (obj) {
                if (x % 2 == 0) {
                    student.name = "鞠婧祎";
                    student.age = 20;
                    student.sex = "female";
                } else {
                    student.name = "斋藤飞鸟";
                    student.age = 18;
                    student.sex = "girl";
                }
                x++;
            }

        }
    }
}
public class GetThread implements Runnable{
    private  Student student;

    public GetThread(Student student) {
        this.student = student;

    }

    @Override
    public void run() {
        while (true) {
            synchronized (SetThread.obj) {
                System.out.println(student.name+":"+student.age+":"+student.sex);

            }

        }
    }
}
public class StudentDemo {
    public static void main(String[] args) {
        Student student = new Student();
        SetThread st = new SetThread(student);
        GetThread gt = new GetThread(student);

        Thread thread = new Thread(st);
        Thread thread1 = new Thread(gt);
        thread.start();
        thread1.start();
    }
}

这样就不会出问题了。

但是，这样只是解决了结果出现人名不对年龄，但还是没解决第一个问题，同一数据连续出现多次，还有空数据，也就是null:0:null数据

A：在生产者还没生产数据时，消费者抢到执行权，这样其实是没有任何意义的，应该等待生产者生产完数据之后消费者再进行消费，

B：在生产者生产完数据后，本该消费者消费数据，但是下一次执行权还是被生产者抢到，这样是有问题的，应该等数据被消费者消费完之后再生产。

正常的思路：

​ A：生产者

​ 先看是否有数据，有就等待，没有就生产，生产完之后唤醒消费者消费

​ B：消费者

​ 先看是否有数据，有就消费，没有就等待 ，等待时唤醒生产者生产

为了解决这一问题，Java提供了一种机制，等待唤醒机制。

这个方法比较特殊，再Object类中，他们是

wait()：等待

notify()：唤醒单个线程

notfyAll()：唤醒全部线程

这几个在Object类中是因为，我们的锁对象是任意锁，而这个方法需要通过锁对象调用，当然锁对象也有可能是Object，所以定义在Objec类中。

撸代码：

public class Student {
    String name;
    int age;
    String sex;
    boolean flag; //默认情况下是false没有数据，如果有数据则为true
}
public class SetThread implements Runnable {
    private Student student;
    private int x= 0;

    public SetThread(Student student) {
        this.student = student;
    }
    @Override
    public void run() {

        while (true) {
            synchronized (student) {

                if (student.flag) {
                    try {
                        student.wait();
                    } catch (InterruptedException e) {
                        e.printStackTrace();
                    }
                }
                if (x % 2 == 0) {
                    student.name = "鞠婧祎";
                    student.age = 20;
                    student.sex = "female";
                } else {
                    student.name = "斋藤飞鸟";
                    student.age = 18;
                    student.sex = "girl";
                }
                x++;

                student.flag = true;
                student.notify();
            }

        }
    }
}

public class GetThread implements Runnable{
    private  Student student;

    public GetThread(Student student) {
        this.student = student;

    }

    @Override
    public void run() {
        while (true) {
            synchronized (student) {
                if (!student.flag) {
                    try {
                        student.wait();
                    } catch (InterruptedException e) {
                        e.printStackTrace();
                    }
                }

                System.out.println(student.name+":"+student.age+":"+student.sex);
                student.flag = false;
                student.notify();
            }

        }
    }
}
public class StudentDemo {
    public static void main(String[] args) {
        Student student = new Student();
        SetThread st = new SetThread(student);
        GetThread gt = new GetThread(student);

        Thread thread = new Thread(st);
        Thread thread1 = new Thread(gt);
        thread.start();
        thread1.start();
    }
}

这段代码运行的结果:

由此可见这样代码写的是正确的，下面来分析下

由代码我们发现，我们开辟了两个线程，thread，thread1，因为线程的运行是随机的我们也不知道，下一次哪个线程开始运行，所以，我们假设thread1线程开始运行，因为thread1线程运行时，开启的线程是消费者，而此时生产者还没有生产，这时消费者线程发现在消费者线程中存在if语句，当进入if语句时，发现生产者没有生产数据，这是程序就在这里阻塞，等待生产者生产数据，因为刚开始x=0 :所以走鞠婧祎的那条if语句，这时，执行到student.notify(),唤醒线程，线程重新回到同步阻塞状态，发现锁可用，且这时消费者发现生产者已经生产数据，所以就继续执行，从而打印出：鞠婧祎:20:female。

二、线程组

线程组：线程组代表一组线程。此外，线程组还可以包括其他线程组。线程组形成一个树，除了初始线程组之外，每个线程组都有一个父进程。允许线程访问有关其线程组的信息，但不能访问有关其线程组的父线程组或任何其他线程组的信息。

我们发现，线程组代表一组线程，可以通过线程组对线程组内的线程进行统一操作，比如设置优先级，设置守护线程等等。

代码样例：

public class MyRunnableDemo {
    public static void main(String[] args) {
        method1();
        method2();
    }

    private static void method2() {
        ThreadGroup tg = new ThreadGroup("郑和惠子");
        MyRunnable myRunnable = new MyRunnable();
        Thread thread = new Thread(tg, myRunnable, "4号窗");
        Thread thread1 = new Thread(tg, myRunnable, "5号窗");
        Thread thread2 = new Thread(tg, myRunnable, "6号窗");

        System.out.println(thread.getThreadGroup().getName());
        System.out.println(thread1.getThreadGroup().getName());
        System.out.println(thread2.getThreadGroup().getName());

        tg.setMaxPriority(10);
        System.out.println(tg.getMaxPriority());
        // thread.start();
        // thread1.start();
        // thread2.start();

    }

    private static void method1() {
        MyRunnable myRunnable = new MyRunnable();
        Thread thread = new Thread(myRunnable, "1号窗");
        Thread thread1 = new Thread(myRunnable, "2号窗");
        Thread thread2 = new Thread(myRunnable, "3号窗");
        System.out.println(thread.getThreadGroup().getName());
        System.out.println(thread1.getThreadGroup().getName());
        System.out.println(thread2.getThreadGroup().getName());
        // thread.start();
        // thread1.start();
        // thread2.start();
    }
}
public class MyRunnable implements Runnable{

    private  int ticket = 200;
    private Object object = new Object();
    @Override
    public void run() {
        while (true) {
            synchronized (object) {
                if (ticket > 0) {
                    try {
                        Thread.sleep(100);
                    } catch (InterruptedException e) {
                        e.printStackTrace();
                    }
                    System.out.println(Thread.currentThread().getName()+"口正在售第"+(ticket--)+"张票");
                }
            }
        }
    }
}

三、生产者消费者代码优化

public class Student {
    private String name;
    private int age;
    private String sex;
    private boolean flag; //默认情况下是false没有数据，如果有数据则为true

    public synchronized void set(String name, int age, String sex) {
        if (this.flag) {
            try {
                this.wait();
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }

        this.name = name;
        this.age = age;
        this.sex = sex;

        this.flag = true;
        this.notify();
    }

    public synchronized void get() {
        if (!this.flag) {
            try {
                this.wait();
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
        System.out.println(this.name+":"+this.age+":"+this.sex);
        this.flag = false;
        this.notify();
    }
}
public class SetThread implements Runnable {
    private Student student;
    private int x = 0;
    public SetThread(Student student) {
        this.student = student;
    }
    @Override
    public void run() {
        while (true) {

            if (x % 2 == 0) {
                student.set("鞠婧祎", 20, "female");
            } else {
                student.set("斋藤飞鸟", 17, "girl");
            }
            x++;
        }
    }
}
public class GetThread implements Runnable {
    private Student student;
    public GetThread(Student student) {
        this.student = student;
    }
    @Override
    public void run() {
        while (true) {
            student.get();
        }
    }
}
public class StudentDemo {
    public static void main(String[] args) {
        Student student = new Student();

        SetThread st = new SetThread(student);
        GetThread gt = new GetThread(student);

        Thread thread = new Thread(st);
        Thread thread1 = new Thread(gt);

        thread.start();
        thread1.start();
    }
}
