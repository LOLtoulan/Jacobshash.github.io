session的理解及使用

会话 ， Session是基于Cookie的一种会话机制。 Cookie是服务器返回一小份数据给客户端，并且存放在客户端上。 Session是，数据存放在服务器端。

    //得到会话ID
    String id = session.getId();

    //存值
    session.setAttribute(name, value);

    //取值
    session.getAttribute(name);

    //移除值
    session.removeAttribute(name);

Session何时创建 ， 何时销毁?

  • 创建

    如果有在servlet里面调用了 request.getSession()

  • 销毁

    session 是存放在服务器的内存中的一份数据。 当然可以持久化. Redis . 即使关了浏览器，session也不会销毁。

    关闭服务器

    session会话时间过期。 有效期过了，默认有效期： 30分钟。

    session默认有效期在tomcat下的conf下的web.xml文件中

    <!-- ==================== Default Session Configuration ================= -->
<!-- You can set the default session timeout (in minutes) for all newly-->
<!-- created sessions by modifying the value below.-->

    <session-config>
        <session-timeout>30</session-timeout>
    </session-config>

Example

public class sessiondemo extends HttpServlet {

    private  static HttpSession session = null;
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {

        resp.setContentType("text/html; charset=UTF-8");

        int id = Integer.parseInt(req.getParameter("id"));

        String[] names = {"Iphone X","XiaoMi 9","IQOO","MeiZu 16S Pro","HuaWeiMate 30Pro","XiaoMi cc9pro"};

        String name = names[id];

        Map<String, Integer> map = (Map<String, Integer>) req.getSession().getAttribute("cart");

        if (map == null) {
            map = new LinkedHashMap<String, Integer>();
            req.getSession().setAttribute("cart", map);
        }

        if (map.containsKey(name)) {
            map.put(name, map.get(name) + 1);
        } else {
            map.put(name, 1);
        }

        resp.getWriter().write("<a href='pro_list.jsp'><h3>继续购物</h3></a><br>");
        resp.getWriter().write("<a href='c.jsp'><h3>去购物车结算</h3></a>");
    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}
public class sessiondemo2 extends HttpServlet {
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        //强制干掉会话，里面存放的任何数据就都没有了。
        HttpSession session = req.getSession();

        session.invalidate();

        //从session中移除某一个数据
        //session.removeAttribute("cart");

        resp.sendRedirect("c.jsp");

    }
    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}
