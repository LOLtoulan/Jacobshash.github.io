JDBC建立连接方式

一、要素一：Driver接口实现类

  1. Driver接口介绍

    • java.sql.Driver接口是所有jdbc接口都要实现的接口，这个接口是提供给数据库厂商使用的，不同的数据库厂商提供不同的实现方式。

    • 在程序中不需要直接去访问实现了Driver的接口类而是由驱动管理器类(java.sql.DriverManager)去调用这些Driver实现

      • Oracle的驱动：oracle.jdbc.driver.OracleDriver
      • MySQL的驱动： com.mysql.jdbc.Driver

      如果我们使用的是maven项目，那么我们就可以直接在pom文件中直接导入jar包，pom中添加如下代码：

      <!-- https://mvnrepository.com/artifact/mysql/mysql-connector-java -->
    <dependency>
      <groupId>mysql</groupId>
      <artifactId>mysql-connector-java</artifactId>
      <version>5.1.38</version>
    </dependency>

      当然我们如果用的不是maven项目，那么我们需要自己进行导包，具体操作网上搜教程。

  2. 加载与注册JDBC驱动

    • 我们把上面的jar包成功导入项目后，我们就需要加载JDBC驱动了。

    • 加载驱动：加载JDBC驱动需要调用Class类的静态方法forName()，通过反射向其传递要加载的JDBC驱动的类名

      • Class.forName(“com.mysql.jdbc.Driver”);

    • 注册驱动：DriverManager类是驱动类的管理类负责驱动程序

      • 使用DriverManager.registerDriver(com.mysql.jdbc.Driver);来注册驱动

      • 但是通常我们不显示调用DriverManager 类的registerDriver()方法来注册驱动程序类的实例，因为 Driver 接口的驱动程序类都包含了静态代码块，在这个静态代码块中，会调用 DriverManager.registerDriver() 方法来注册自身的一个实例。下图是MySQL的Driver实现类的源码：

        public class Driver extends NonRegisteringDriver implements java.sql.Driver {
    //
    // Register ourselves with the DriverManager
    //
    static {
        try {
            java.sql.DriverManager.registerDriver(new Driver());
        } catch (SQLException E) {
            throw new RuntimeException("Can't register driver!");
        }
    }

    /**
     * Construct a new driver and register it with DriverManager
     * 
     * @throws SQLException
     *             if a database error occurs.
     */
    public Driver() throws SQLException {
        // Required for Class.forName().newInstance()
    }
}

二、要素二：URL

  • JDBC URL 用于标识一个被注册的驱动程序，驱动程序管理器通过这个 URL 选择正确的驱动程序，从而建立到数据库的连接。

  • JDBC URL的标准由三部分组成，各部分间用冒号分隔。

    • jdbc:子协议:子名称
    • 协议：JDBC URL中的协议总是jdbc
    • 子协议：子协议用于标识一个数据库驱动程序
    • 子名称：一种标识数据库的方法。子名称可以依不同的子协议而变化，用子名称的目的是为了定位数据库提供足够的信息。包含主机名(对应服务端的ip地址)，端口号，数据库名

  • 举例：

  • 几种常用数据库的 JDBC URL

    • MySQL的连接URL编写方式：

      • jdbc:mysql://主机名称:mysql服务端口号/数据库名称?参数=值&参数=值
      • jdbc:mysql://localhost:3306/test
      • jdbc:mysql://localhost:3306/test?useUnicode=true&characterEncoding=utf8（如果JDBC程序与服务器端的字符集不一致，会导致乱码，那么可以通过参数指定服务器端的字符集）
      • jdbc:mysql://localhost:3306/test?user=root&password=123456

    • Oracle 9i的连接URL编写方式：

      • jdbc:oracle:thin:@主机名称:oracle服务端口号:数据库名称
      • jdbc:oracle:thin:@localhost:1521:atguigu

    • SQLServer的连接URL编写方式：

      • jdbc:sqlserver://主机名称:sqlserver服务端口号:DatabaseName=数据库名称

      • jdbc:sqlserver://localhost:1433:DatabaseName=atguigu

三、要素三：用户名和密码

  • user,password可以用“属性名=属性值”方式告诉数据库
  • 可以调用 DriverManager 类的 getConnection() 方法建立到数据库的连接

四、数据库连接方式举例

4.1连接方式一

通过调用第三方API来进行创建连接数据库对象

/*
创建连接步骤：
1.创建驱动对象（Driver）
2.获取url（url连接数据库的url）
3.获取user和password（MySQL的user和password）
4.创建Connection对象（调用driver的connect对象）
 */
public class ConnectionTest {
    @Test
    public void testConnection1() throws SQLException, ClassNotFoundException {
        Driver driver = new com.mysql.jdbc.Driver();

        //http://www.loltoulan.top/
        //jdbc:mysql:  代表协议，和上面URL中的http:含义差不多
        //localhost:3306 代表ip地址和端口号
        //test  是表名，表示我们要对test表进行操作
        String url = "jdbc:mysql://localhost:3306/test";

        //将用户名和密码封装到properties里面
        Properties info = new Properties();
        //用户名
        info.setProperty("user", "abc");
        //密码
        info.setProperty("password", "abc");

        Connection connect = driver.connect(url, info);
        System.out.println(connect);
    }
}

4.2连接方式二

不调用第三方API来实现来连接，有更好的可移植性

/*
通过反射来进行操作
1.通过反射调用Class的forName方法来进行获取mysql的driver驱动
2.调用class的newInstance方法获取class对象forName方法里面的类的对象并强转为Driver
3.获取url（url连接数据库的url）
4.获取user和password（MySQL的user和password）
5.创建Connection对象（调用driver的connect对象）
 */
public class ConnectionTest {
    @Test
    public void testConnection2() throws Exception{
       //通过反射来实现
        Class clazz = Class.forName("com.mysql.jdbc.Driver");
        Driver driver = (Driver) clazz.newInstance();

        String url = "jdbc:mysql://localhost:3306/test";
        Properties info = new Properties();
        info.setProperty("user", "root");
        info.setProperty("password", "root");
        Connection connect = driver.connect(url, info);
        System.out.println(connect);
    }
}

运行该测试类，如果控制台上出现如下图所示，即为连接成功：

4.3连接方式三

使用DriverManager替换Driver

/*
使用DriverManager来替换Driver
1.通过反射调用Class的forName方法来进行获取mysql的driver驱动
2.调用class的newInstance方法获取class对象forName方法里面的类的对象并强转为Driver
3.获取url（url连接数据库的url）
4.获取user和password（MySQL的user和password）
5.注册驱动，调用DriverManager的registerDriver来注册驱动
6.使用DriverManager.getConnection(url, user, password)来创建连接
 */
public class ConnectionTest {
    @Test
    public void testConnection3() throws Exception {
        Class clazz = Class.forName("com.mysql.jdbc.Driver");
        Driver driver = (Driver) clazz.newInstance();
        DriverManager.registerDriver(driver);
        String url = "jdbc:mysql://localhost:3306/test";
        String user = "root";
        String password = "root";
        Connection connection = DriverManager.getConnection(url, user, password);
        System.out.println(connection);
    }
}

4.4连接方式四

优化第三种方式

 /*
     1.获取url（url连接数据库的url）
     2.获取user和password（MySQL的user和password）
     3.通过反射加载Mysql驱动加载类
     4.使用DriverManager.getConnection(url, user, password)来创建连接
     */
public class ConnectionTest {
    @Test
    public void testConnection4() throws Exception {

        String url = "jdbc:mysql://localhost:3306/test";
        String user = "root";
        String password = "root";
        //这里不需要注册驱动的原因是我们进入com.mysql.jdbc.Driver类中发现其实他已经帮我们把驱动给注册了
        Class.forName("com.mysql.jdbc.Driver");

        Connection connection = DriverManager.getConnection(url, user, password);
        System.out.println(connection);
    }
}

4.5连接方式五

最终版

好处：
1.数据与代码分离，实现解耦

2.如果需要修改，只修改配置文件，可以避免面重复打包

/*
1.获取Properties对象
2.通过类加载器获取文件的路径
3.通过properties.load方法来对文件进行加载
4.通过properties.getProperty(key);方法获取配置文件中的value值
5.通过反射调用Class的forName方法来进行获取mysql的driver驱动
6.使用DriverManager.getConnection(url, user, password)来创建连接
*/
public class ConnectionTest {
    @Test
    public void testConnection5() throws Exception {

        Properties properties = new Properties();
        //类加载器，通过类加载器来获取路径流，来获取文件对象，默认的识别路径为src下
        InputStream inputStream = ConnectionTest.class.getClassLoader().getResourceAsStream("jdbc.properties");
        properties.load(inputStream);
        String url = properties.getProperty("url");
        String user = properties.getProperty("user");
        String password = properties.getProperty("password");

        Class.forName(properties.getProperty("driverClass"));

        Connection connection = DriverManager.getConnection(url, user, password);
        System.out.println(connection);
    }
}
user = root
password = root
url = jdbc:mysql://localhost:3306/test
driverClass = com.mysql.jdbc.Driver
