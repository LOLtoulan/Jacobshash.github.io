jQuery与Ajax结合

一、jQuery是什么？

JQuery 是 javascript 的一个框架集合 ， 并不是什么新的技术。

二、作用

JQuery 是一个 javaScrip的框架，它的宗旨是： Write Less , Do More. 写更少的代码，完成更多的工作。 平常我们如果使用ajax来完成交互工作，可能要写很多代码。并且代码重复性比较高。

那么能不能让代码写的少一点，重复性第一点 , jQuery 做的就是这个事。

三、用jQuery写Ajax代码

function post() {

    $.post("/day16/DemoServlet02", {name:"zhangsan",age:18},function(data,status) {

public class DemoServlet03 extends HttpServlet {

    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        resp.setContentType("text/html; charset=UTF-8");
        resp.setHeader("Access-Control-Allow-Origin","*");
        String name = req.getParameter("name");
        int age = Integer.parseInt(req.getParameter("age"));
        System.out.println("我收到一个get请求" + name+" : "+age);
        resp.getWriter().write("给你一份数据"+name+" : "+age);
    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        resp.setContentType("text/html; charset=UTF-8");
        resp.setHeader("Access-Control-Allow-Origin","*");
        System.out.println("我收到一个post请求，现在要去get方法了");
        doGet(req,resp);
    }
}
