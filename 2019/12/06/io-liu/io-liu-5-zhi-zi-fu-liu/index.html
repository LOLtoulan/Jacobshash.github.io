值得注意的是：使用不同的编码格式打开文件，看的信息可能不一样

close();和flush();区别

close()之后无法再写数据

flush();之后还可再写数据

三、字符流读数据

和字节流读数据一样

public class ZiFuLiuDemo1 {
    public static void main(String[] args) throws IOException {

        //指定字符集
        InputStreamReader inputStreamReader = new InputStreamReader(new FileInputStream("b.txt"));
        //默认字符集
        //InputStreamReader inputStreamReader = new InputStreamReader(new FileInputStream("b.txt"));

        //一次读取一个字符
        /*int len = 0;
        while((len = inputStreamReader.read())!=-1) {
            System.out.print((char)len);
        }*/


        //一次读取自已字符数组
        char[] chars = new char[1024];
        int len = 0;
        while ((len = inputStreamReader.read(chars)) != -1) {
            System.out.println(new String(chars,0,len));
        }
        inputStreamReader.close();
    }
}

四、OutputStreamWriter和InputStreamReader复制文件

public class OutputStreamWriterDemo1 {
    public static void main(String[] args) throws IOException {


        //method1();
        //method2();


    }

    private static void method2() throws IOException{
        OutputStreamWriter outputStreamWriter = new OutputStreamWriter(new FileOutputStream("la.txt"));
        InputStreamReader inputStreamReader = new InputStreamReader(new FileInputStream("b.txt"));

        char[] chars = new char[1024];
        int len = 0;

        while ((len = inputStreamReader.read(chars)) != -1) {
            outputStreamWriter.write(chars,0,len);
        }


        outputStreamWriter.close();
        inputStreamReader.close();
        System.out.println("复制完成！");
    }

    private static void method1() throws IOException{
        OutputStreamWriter outputStreamWriter = new OutputStreamWriter(new FileOutputStream("lala.txt"));
        InputStreamReader inputStreamReader = new InputStreamReader(new FileInputStream("b.txt"));

        int len = 0;

        while ((len = inputStreamReader.read()) != -1) {
            outputStreamWriter.write(len);
        }

        outputStreamWriter.close();
        inputStreamReader.close();
        System.out.println("复制完成！");
    }

}

九、总结

字符流 = 字节流 + 编码表

编码表：编码表就是由字符和对相应的数值组成的一张表

常见的编码表：

  • ASCII
  • ISO-8859-1
  • GB2312
  • GBK
  • GB18030
  • UTF-8

字符串中的编码问题

  • 编码：String – byte[]
  • 解码：byte[] – String

IO流中的编码问题

  • OutputStreamWriter
    • OutputStreamWriter(OutputStream os)
    • OutputStreamWriter(OutputStream os , String charsetName)
  • InputStreamReader
    • InputStreamReader(InputStream is)
    • InputStreamReader(InputStream is , String charsetName)
  • 编码一致即可

字符流

  • Rreader
    • InputStreamReader
      • FIleReader
    • BufferReader
  • Writer
    • OutputStreamWriter
      • FileWriter
    • BufferWriter
