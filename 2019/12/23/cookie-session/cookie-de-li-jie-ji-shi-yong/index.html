cookie的理解及使用

一、什么是cookie

Cookie直译过来就是”饼干，小甜点”，但是Cookie在我们这里表示的是一份小数据，是服务器给客户端的，并且存储在客户端的小数据。

二、有什么用

自动登录、浏览记录、购物车。

三、为什么要有Cookie

http请求是无状态的。什么叫无状态，打个比方，我们今天在浏览器上登录了一个网站，然后我们将浏览器关闭，下次我们再登录，我们登陆网站，还要输入用户名，密码，也就是说，我们一旦关闭浏览器，再打开浏览器，浏览器就不认识我们了，这就叫无状态。

为了更好的用户体验，更好的交互【自动登录】，从大公司层面就是更好的手机用户体验

简单使用

public class mycookie extends HttpServlet {
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {

        resp.setContentType("text/html; charset=UTF-8");
        //不能写中文
        Cookie cookie = new Cookie("beauty", "ZhengHeHuiZi");

        resp.addCookie(cookie);

        resp.getWriter().write("请求成功");
        resp.getWriter().println("请求成功");
    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}

四、Cookie的简单使用

public class mycookie2 extends HttpServlet {
    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {

        //遍历cookie
        Cookie[] cookies = req.getCookies();
        if (cookies != null) {
            for (Cookie cookie : cookies) {
                System.out.println(cookie.getName()+" = "+cookie.getValue());
            }
        }

        resp.setContentType("text/html; charset=UTF-8");
        Cookie cookie = new Cookie("beauty", "ZhengHeHuiZi");
        Cookie cookie1 = new Cookie("nationality", "China");

        //设置cookie的有效期
        //这个cookie的有效期。  默认情况下 ，

        //关闭浏览器后，cookie就没有了。 ---> 针对没有设置cookie的有效期。
        //expiry： 有效 以秒计算。
        //正值 ： 表示 在这个数字过后，cookie将会失效。
        //负值： 关闭浏览器，那么cookie就失效， 默认值是 -1
        cookie.setMaxAge(60 * 60 * 24 * 7);
        //给当前value赋新的值，一般不用
        //cookie.setValue("haah");
        //用于指定只有请求了指定的域名，才会带上该cookie
        cookie.setDomain(".LOLtoulan.cn");

        //只有访问该域名下的cookieDemo的这个路径地址才会带cookie
        cookie.setPath("/mycookie2");

        resp.addCookie(cookie);
        resp.addCookie(cookie1);

        resp.getWriter().println("请求成功");
    }

    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}

五、案例一（获取登录时间）

public class myservlet extends HttpServlet {

    @Override
    protected void doGet(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        String name = req.getParameter("name");
        String password = req.getParameter("password");
        resp.setContentType("text/html; charset=UTF-8");
        System.out.println(name+":"+password);
        if (name.equals("abc") && password.equals("123")) {
            Cookie[] cookies = req.getCookies();

            Cookie last = CookieUtil.findCookie(cookies, "last");
            long time = System.currentTimeMillis();
            if (last == null) {
                Cookie cookie = new Cookie("last",  time+ "");
                cookie.setMaxAge(60 * 60);
                resp.addCookie(cookie);
                resp.getWriter().println("登陆成功，欢迎您，您是首次登陆，登陆时间为"+new Date(time));
            } else {
                Long value = Long.valueOf(last.getValue());
                resp.getWriter().println("登陆成功，欢迎您，您上次登陆时间为"+new Date(value));
                last.setValue(System.currentTimeMillis()+"");
                resp.addCookie(last);
            }
        } else {
            resp.getWriter().println("账号或密码错误");
        }
    }
    @Override
    protected void doPost(HttpServletRequest req, HttpServletResponse resp) throws ServletException, IOException {
        doGet(req, resp);
    }
}

六、总结

  1. 服务器给客户端发送过来的一小份数据，并且存放在客户端上。

  2. 获取cookie， 添加cookie

    request.getCookie();

    response.addCookie();

  3. Cookie分类

    会话Cookie

    默认情况下，关闭了浏览器，那么cookie就会消失。

    持久Cookie

    在一定时间内，都有效，并且会保存在客户端上。

    cookie.setMaxAge(0); //设置立即删除

    cookie.setMaxAge(100); //100 秒

  4. Cookie的安全问题。

由于Cookie会保存在客户端上，所以有安全隐患问题。 还有一个问题， Cookie的大小与个数有限制。 为了解决这个问题 —> Session .
